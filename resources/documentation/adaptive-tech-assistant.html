üîÑ Cognitive Loop Implementation

Example: Content Summarization Cognitive Loop

  1. Planner: "Summarize this AI research paper into intro, body, conclusion"
  2. Executor: Generates initial summaries for each section
  3. Verifier: Checks coverage, accuracy, key points inclusion
  4. Critic: "Intro too technical, body missing key algorithm details"
  5. Executor: Revises based on critic feedback
  6. Loop continues: Until quality threshold met or max iterations reached
